# Tips

Useful Hot Key

## Editor

* Ctrl + D: cut one line
* Alt + Shift + A: block edit
* Shift + Enter / Ctrl + Shift + Enter: vim 'o' / 'O'
* Alt + up/down: move line.
* Ctrl + Alt + up/down: copy line

## Search & Jump

* `F3` go to definition
* `Ctrl + Shift + R` **file** level search
