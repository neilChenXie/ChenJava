# Basic

```bash
#####Clean####
#Clean war package
mvn clean

#####Eclipse#####
#Create Eclipse specific file Before importing
mvn eclipse:eclipse

#Clean eclipse related files
mvn eclipse:clean

#####After Write(Not Eclipse related Independent)#####
#Compile
mvn compile

#Test
mvn test

#Create war package
mvn package
```
